ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА ДЛЯ КОНВЕРТАЦИИ MKV В MP4

АВТОМАТИЧЕСКАЯ УСТАНОВКА И ЗАПУСК (РЕКОМЕНДУЕТСЯ):
1. Дважды кликните на файл "ЗАПУСТИТЬ_БОТА.bat" в проводнике Windows
2. Скрипт автоматически:
   - Установит все необходимые зависимости Python
   - Скачает и установит FFmpeg
   - Добавит FFmpeg в PATH
   - Запустит бота
3. Если появится предупреждение безопасности Windows, нажмите "Подробнее" и "Выполнить в любом случае"

ЗАПУСК С АВТОМАТИЧЕСКИМ ДОБАВЛЕНИЕМ FFMPEG В PATH (ЕСЛИ ВОЗНИКАЮТ ОШИБКИ):
1. Дважды кликните на файл "run_with_ffmpeg_path.bat" в проводнике Windows
2. Этот скрипт:
   - Проверит наличие Python и FFmpeg
   - Установит FFmpeg, если он не найден
   - Добавит FFmpeg в PATH для текущей сессии
   - Запустит бота
3. Используйте этот способ, если при конвертации возникает ошибка "Не удается найти указанный файл"

АЛЬТЕРНАТИВНЫЕ СПОСОБЫ ЗАПУСКА:

1. Запуск бота без проверки FFmpeg:
   - Дважды кликните на файл "run_bot.bat" в проводнике Windows
   - Бот запустится даже если FFmpeg не установлен (но конвертация работать не будет)

2. Установка FFmpeg вручную:
   - Перейдите на сайт: https://ffmpeg.org/download.html
   - В разделе "Windows" выберите "Windows builds from gyan.dev"
   - Скачайте файл "ffmpeg-release-essentials.zip"
   - Создайте папку C:\ffmpeg
   - Распакуйте содержимое архива в эту папку
   - Скопируйте содержимое папки bin из распакованной папки в C:\ffmpeg\bin
   - Добавьте C:\ffmpeg\bin в переменную PATH системы
   - Перезапустите компьютер или откройте новое окно командной строки

ПРОВЕРКА РАБОТЫ БОТА:
1. Найдите своего бота в Telegram
2. Отправьте команду /start
3. Отправьте файл .mkv для конвертации
