ИНСТРУКЦИЯ ПО РУЧНОЙ УСТАНОВКЕ FFMPEG

1. Скачайте FFmpeg для Windows:
   - Перейдите на сайт: https://ffmpeg.org/download.html
   - В разделе "Windows" выберите "Windows builds from gyan.dev"
   - Скачайте файл "ffmpeg-release-essentials.zip"

2. Распакуйте архив:
   - Создайте папку C:\ffmpeg
   - Распакуйте содержимое архива в эту папку
   - Внутри будет папка с названием вроде "ffmpeg-6.1.1-essentials_build"
   - Скопируйте содержимое папки bin из этой папки в C:\ffmpeg\bin

3. Добавьте FFmpeg в переменную PATH:
   - Нажмите Win + X и выберите "Система"
   - Нажмите "Дополнительные параметры системы"
   - Нажмите "Переменные среды"
   - В разделе "Системные переменные" найдите переменную "Path" и нажмите "Изменить"
   - Нажмите "Создать" и добавьте путь: C:\ffmpeg\bin
   - Нажмите "ОК" во всех окнах

4. Проверьте установку:
   - Откройте новое окно командной строки (cmd) или PowerShell
   - Введите команду: ffmpeg -version
   - Если вы видите информацию о версии FFmpeg, значит установка прошла успешно

5. Запустите бота:
   - Перейдите в папку с ботом
   - Запустите команду: python bot.py
